# **在线 KTV 音乐可视化播放器**
一款基于Web 前端 + node.js 轻量接口开发的沉浸式在线点歌播放系统，支持音乐可视化动态背景、实时歌词同步、点歌列表管理，可直接部署于 Vercel 等静态 托管平台，开箱即用。
此版本为 Vercel托管版本，无数据库，使用 Vercel的KV存储，使用外联数据库请使用PHP架构版本，KTV-MP3
## 核心功能
多风格可视化动态背景：粒子波动、频谱柱状、渐变流动等 6 种音频律动动效，随音乐节奏实时响应
## 精准歌词逐字高亮：时间轴对齐，当前句高亮、前后句预览，媲美原生 KTV 体验
完整点歌控制台：播放 / 暂停 / 切歌、音量调节、列表点歌、置顶 / 上移 / 下移 / 删除
列表快捷操作：点击直接播放指定歌曲，按钮布局美观易用
首页扫码入口：左上角自动生成当前站点二维码，手机扫码快速进入
自适应全屏布局：PC 端沉浸式体验，支持窗口大小自适应
## **技术栈**
前端：HTML5 Canvas、原生 JavaScript、CSS3、Web Audio API
后端：node.js（轻量接口）

## 特点
界面炫酷、动效流畅，低资源占用
接口轻量、无复杂依赖，部署极简单
模块化背景脚本，可自由扩展新动效
适合个人使用、展示或二次开发
## 超清晰一步到位教程：Vercel Blob Store 接入项目 + 自动初始化数据
我给你最简单、不用手动建文件、自动初始化的完整方案，你跟着做 3 分钟搞定！
# 一、先做 2 步准备（必须）
1. 打开 Vercel 控制台 → 你的项目 → Storage → Create Blob
创建一个 Blob 存储（免费，足够用）
2. 把 Blob 连接到项目
Vercel 会自动提示：Connect to project → 点一下，自动添加环境变量：BLOB_READ_WRITE_TOKEN
这就叫连接完成！不用任何其他配置！
